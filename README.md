# FreshMart – Premium Grocery E-commerce

A modern, high-performance grocery shopping interface built with Vanilla HTML, CSS, and JavaScript.

## 🎨 Design System

- **Colors**:
  - Deep Blue (`#01406D`) - Primary Brand
  - Teal (`#01B4BA`) - Accents & Secondary
  - Off White (`#F5FEFE`) - Background
  - Orange (`#FF7A0F`) - CTA & Highlights
- **Typography**:
  - **Poppins**: For headings and brand elements (Modern & Premium)
  - **Inter**: For clean, readable body text
- **Aesthetics**: Glassmorphism effects, smooth hover transitions, and a responsive grid layout.

## 🚀 Features

- **Dynamic Product Grid**: Filterable items with price tags and ratings.
- **Interactive Cart**: Sidebar drawer with real-time tax/total calculation.
- **Hero Banner**: High-impact messaging with floating visual elements.
- **Category Navigation**: Quick-access chips for easy browsing.
- **Responsive Design**: Optimized for Desktop and Mobile.

## 📂 Project Structure

- `index.html`: Main semantic structure and SEO tags.
- `style.css`: Comprehensive styling and design tokens.
- `app.js`: Business logic, product data, and cart management.

## 🛠️ Getting Started

Simply open `index.html` in any modern browser to view the application.
